## 🚀 Getting Started

1. **Clone the repository**
```bash
   git clone https://github.com/YUVAN-cse/Frontend-mini-projects.git
```

2. **Navigate to any project folder**
```bash
   cd Frontend-mini-projects/Calculator
```

3. **Open `index.html` in your browser**
```bash
   open index.html
```
   > For React projects, run `npm install` and `npm start`
